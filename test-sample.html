Update Image Size for Source Tag - テスト用サンプル

例1: srcset属性を持つsourceタグ

例2: src属性を持つsourceタグ

例3: 既にwidth/height属性があるsourceタグ

使い方

  1. 上記の<source>タグ内にカーソルを置く
  2. Cmd+Shift+I（Mac）またはCtrl+Shift+I（Windows/Linux）を押す
  3. または、コマンドパレット（Cmd+Shift+P / Ctrl+Shift+P）から「Update Image Size for Source Tag」を選択
  4. 画像のwidth/height属性が自動的に追加または更新されます

注意: 実際に画像ファイルを用意して、パスを正しく設定してください。
