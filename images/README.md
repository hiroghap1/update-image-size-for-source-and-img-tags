# テスト用画像ディレクトリ

このディレクトリにテスト用の画像ファイルを配置してください。

## 必要な画像ファイル

`test-sample.html`をテストするには、以下のファイル名で画像を配置してください：

- `sample.webp` - WebP形式の画像
- `sample.jpg` - JPEG形式の画像
- `photo.png` - PNG形式の画像（大きいサイズ）
- `photo-small.png` - PNG形式の画像（小さいサイズ）
- `banner.jpg` - JPEG形式のバナー画像

任意の画像ファイルをこれらの名前で配置すれば、拡張機能のテストができます。
